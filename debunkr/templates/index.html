How It Works.

Multi-layer AI verification pipeline from raw text to verdict.

01 —

Text Preprocessing

URLs removed, lowercased, lemmatised, stopwords stripped — identical pipeline to model training.

02 —

TF-IDF + LSA

10,000 TF-IDF features reduced to 300 dimensions via Latent Semantic Analysis (TruncatedSVD).

03 —

Word2Vec Embeddings

100-dimensional semantic embeddings combined with LSA into a 400-dim feature vector.

04 —

XGBoost Classification

95% test accuracy. Confidence extracted from class probability distribution.

05 —

Google Fact Check API

Cross-referenced against millions of reviewed claims from professional fact-checkers worldwide.

06 —

Snopes + FactCheck.org

Live search and rating extraction from two of the world's most trusted fact-checking organisations.

Verdict Logic

The ML model carries 2× weight. Each external source gets 1 vote. The majority wins — returning REAL, FAKE, or UNCERTAIN with a confidence percentage.

Input Formats

Raw text or tweets, article URLs (auto-fetched), and file uploads (.txt, .csv). All run through the exact same pipeline.
